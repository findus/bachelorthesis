\section{Anwendungsfälle}

Ein Anwendungsfall ist eine Abstraktion, die alle möglichen Szenarien beschreiibt, die die beschriebene Funktionalität enthalten.\cite[S. 76]{BrueggeDutoit}

Ein Szenario ist eine Instanz eines Anwendungsfalls; wenn wir diesen Anwendungsfall identifizieren, haben wir damit alle
möglichen Szenarien für eine gegebene Funktionalität.\cite[S. 160]{BrueggeDutoit}

\subsection{Anwendungsfall 1: Server Starten}
Der Spieler startet den Server auf einem lokalen Computer. Anschließend wählt er eine virtuelle Spielwelt aus und lässt den Server diese laden.

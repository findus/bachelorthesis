\section{Anforderungen}
\label{requirements}



\subsection{Spielmodus und Spielziel} 
Das neue Spiel muss einen Spielmodus haben, dessen Spielziel ganz ohne Anwendung von Gewalt zu erreichen ist.
Außerdem soll der Spielmodus teambasiert sein und Kooperation der Spieler ermöglichen oder bestenfalls sogar fordern.


\subsection{Direkte Einwirkung auf den Gegner}
Das Spiel muss dem Spieler die Möglichkeit bieten, direkt auf den Avatar eines Gegenspielers einzuwirken. Diese
Einwirkung muss zeitlich und örtlich direkt sein in einer Weise, dass der Spieler den Avatar des Gegenspielers
anvisieren muss und auf eine bestimmte Eingabe eine direkte Rückmeldung bekommt.


\subsection{Spezielle Aktion}
Das Spiel muss dem Spieler die Möglichkeit bieten, eine besondere Aktion auszuführen, die einen besonders starken
behindernden Einfluss auf einen Gegner bewirkt, wenn er es schafft eine schwer zu erreichende Konstellation im
Spielgeschehen herbeizuführen.


\subsection{Karte wechseln}
Das Spiel muss dem Administrator die Möglichkeit bieten, verschiedene Spielwelten zu laden.

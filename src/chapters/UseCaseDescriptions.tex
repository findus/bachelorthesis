\chapter{Detaillierte Beschreibung der Anwendungsfälle}
\label{chapter:UseCaseDescriptions}

\section{Anwendungsfall 1: Server Starten}
Der Administrator startet den Server auf einem (lokalen) Computer.

\section{Anwendungsfall 2: Map laden}
Der Administrator wählt er eine virtuelle Spielwelt aus und lässt den Server diese laden.

\section{Anwendungsfall 2: Server beenden}
Der Administrator beendet den laufenden Server.

\section{Anwendungsfall 3: Spiel beitreten}
Der Spieler möchte einem Spiel beitreten. Dazu startet er das Spiel. Er wählt einen Server aus. Falls der gewünschte
Server nicht verfügbar ist, kann er ihn manuel hinzufügen. Der Spieler wählt ein Team aus und beginnt zu spielen.

\section{Anwendungsfall 4: Spiel verlassen}
Der Spieler verlässt das Spiel.
